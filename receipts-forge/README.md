# AetherTwin Receipts Forge

Turn scattered work into proof.

A local-first React/Vite prototype that converts project traces into reusable proof-of-work cards: portfolio blurbs, X posts, README sections, job-search bullets, and devlog entries.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Create proof receipts from notes, repos, demos, screenshots, posts, and devlogs
- Save receipts to localStorage
- Browse and filter the receipt library by project
- Generate X posts, X threads, portfolio blurbs, README sections, job bullets, and devlogs
- Copy generated outputs
- Export all receipts as Markdown
- View project timelines
- Starter receipts for Fairy OS, QI-Games, AetherTwin Helm, and Magic Meets Machine

## v0.1 constraints

- no backend
- no auth
- no real AI API calls
- no scraping
- no external posting
- no database beyond localStorage

No tax. Only trace.
